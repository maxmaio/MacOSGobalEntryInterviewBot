## Global Entry Interview Finder for MacOS

### Description

When running, this bot will poll the global entry endpoint for available appointments at a specified location. When an 
appointment becomes available it will push a convenient MacOS notification.

#### Example
![screenshot](./images/screenshot.jpg)
==============================

### Instructions
- make sure node.js is installed
- clone this repo
- add your `authToken` and `locationId` to `/index.js`
- run `npm install` in your terminal
- run `node index.js` in your terminal and keep it running on your machine
- when an appointment becomes available a notification will appear!
